# Go / No-Go Game

A minimal Go/No-Go web page with 3-second stages, 90/10 Go vs No-Go bias, and reaction time logging.

## Run locally

Using npx (no install):

```
npx http-server .
```

Then open the printed URL (default http://localhost:8080).

Alternate (after installing once):

```
npm install -g serve
serve .
```

Open http://localhost:3000 (or the URL shown). Stop with Ctrl+C.
